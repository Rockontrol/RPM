Project Manager & Rockontrol Tools
====

简介：
主要应用于罗克佳华公司员工使用。

主要应用于：
A：任务跟踪和任务分解
公司主要采用绩效管理方式，以组织目标为核心，进行月度、季度、年度绩效考核。定制目标 -> 转换为相关任务 -> 细化分解任务 --> 完成任务，达到目标。
所以，对于工作内容可以采用目标+任务的方式来进行管理跟踪。主要对工作的目标或任务进行登记，并对目标或者任务进行细分和设置主要节点，比如：项目第一阶段、项目第二阶段、测试节点、出1.0版、2.0截止等自定义节点。

B：VP报工
公司每月主要采用VisualProject进行组织项目管理，主要是进行项目的完成度跟踪和员工的工作内容管理。需要员工每天进行报工登记和工作周记并由上级审核管理。

C：OA办公
主要兼容OA代办通知功能，可以及时通知到个人，快速处理加速流转流程。

D：RTX通讯
兼容RTX通讯协议。

项目设计规划：
A：任务分解跟踪功能主要通过日历功能实现，主要通过日历视图来展示任务信息。
B：数据层任务分解管理模型，通过组织数据来完成业务模型。
